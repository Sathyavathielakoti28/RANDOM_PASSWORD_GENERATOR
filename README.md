# 🔐 Random Password Generator

A simple and secure **Random Password Generator** built using **Python** and **Tkinter**. This application allows users to generate strong passwords based on their preferred length and selected character types.

## 📌 Features

- Generate secure random passwords
- Select password length
- Include:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Symbols (!@#$%^&*)
- Input validation
- Copy generated password to clipboard
- Simple and beginner-friendly GUI
- Modular project structure

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)
- string module
- secrets module

## 📖 How to Use

1. Enter the desired password length.
2. Select the character types you want to include.
3. Click **Generate Password**.
4. The generated password will appear in the text box.
5. Click **Copy Password** to copy it to the clipboard.
